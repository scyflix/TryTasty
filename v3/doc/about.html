Story

TryTasty.de was first created as a learning project to practice basic HTML and CSS learnt at the time. With time, TryTasty has been evolving from what it used to be. It all started with the first deployment on June 17, 2025, which featured a simple homepage with just a few recipes. The layout and design were basic. No responsiveness, only the foundational HTML and CSS learnt at that time. Since then, it's continuously being updated and improved with new ideas and feedback. This project remains one that is actively developed and maintained.

Version Updates

Below are lists and descriptions of updates made to TryTasty.

Version: 1.0.0(Initial Release)

11.08.2025

URL: TryTasty V1

  • Colour Change: Updated from white and orange to beige, orange and orangered.
  • Login/Signup System: Added login functionality to prepare for future expansion and user personalization.
  • Videos Page: Introduced a new page for users to watch our food videos directly on the site.
  • Header Layout Update: Changed the navigation layout from a column with a logo to a row-based design, removing the logo to make room for additional links like Login.
  • Page Access Restrictions: Limited access to certain pages (e.g., Recipe Submission and Videos) to logged-in users only.
  • User Experience (UX) Improvements: Enhanced SEO and accessibility for better usability and user comfort.

These changes helped improve the overall user experience and gave TryTasty a more structured direction.

Version 2.0.0

05.09.2025

URL: TryTasty V2

  • Default Theme Update: Updated from beige, orange and redorange to a radial gradient of yellow and orange.
  • Basic Bot: Introduced a recipe search bot to help users find recipes easily.
  • Improved Performance: Optimised image by converting them to AVIF, icreasing site loading speed.
  • Sidebar: Moved navigation links from header to the sidebar due to increase in links, for easier navigation between pages.
  • Favorites Feature: Introduced an "Add to favorites" option for users to save recipes for later. Saved recipes can be found in the favorites Page.
  • Legal Page: Added essential legal documents, including Imprint and Privacy Policy.

These changes shows that TryTasty is moving towards a professional web app that cares for its users and hope to give users a good experience.

Version 3.0.0(Current version)

28.12.2025

URL: TryTasty V3

  • Profiles page:Introduced the profiles page where users can see the name, email they logged in with, number of recipes they saved(favorites) and number of recipes they have submited through the submit page.
  • Unique recipe URL: Gave each recipe a unique URL for easy recipe sharing.
  • Toast Navigation: Added a toast navigation for mobile to give the platform an App-like feel for good user experience.
  • Light and Dark mode: Added light and dark mode functionality, giving users the ability to switch interface with the click of a button.
  • About page: Added an about page to make user learn more about TryTasty and build trust.
